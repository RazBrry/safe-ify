## Instructions

1. Create `internal/config/global_test.go`:
   - `TestLoadGlobal_ValidConfig` -- create temp file with valid YAML, set 0600, load, verify struct.
   - `TestLoadGlobal_MissingFile` -- non-existent path, verify ConfigNotFoundError.
   - `TestLoadGlobal_InsecurePermissions` -- create temp file with 0644, verify ConfigInsecureError.
   - `TestSaveGlobal_CreatesWithCorrectPermissions` -- save to temp dir, stat file, verify 0600.
   - `TestSaveGlobal_CreatesDirectory` -- save to non-existent dir, verify dir created with 0700.

2. Create `internal/config/project_test.go`:
   - `TestLoadProject_ValidConfig` -- create temp file with valid project YAML, load, verify.
   - `TestFindProjectConfig_CurrentDir` -- create `.safe-ify.yaml` in temp dir, find from same dir.
   - `TestFindProjectConfig_ParentDir` -- create `.safe-ify.yaml` in parent, find from child dir.
   - `TestFindProjectConfig_NotFound` -- empty temp dir tree, verify ProjectConfigNotFoundError.

3. Create test fixtures in `testdata/`:
   - `testdata/valid-global-config.yaml` -- sample global config.
   - `testdata/valid-project-config.yaml` -- sample project config.

4. Run all tests: `go test ./internal/config/ -v -race`
5. Verify all tests pass.
