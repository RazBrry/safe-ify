## Instructions

1. Add to `internal/tui/forms.go`:
   - `InitSelectInstanceForm(instances []string)` -- returns `huh.Select` for choosing an instance.
   - `InitSelectAppForm(apps []AppOption)` -- returns `huh.Select` for choosing an application (display: name + UUID).
   - `InitPermissionsForm()` -- returns `huh.MultiSelect` with all agent commands. Selected items are DENIED.
   - Define `AppOption` struct with `Name` and `UUID` fields.

2. Create `internal/cli/init.go`:
   - Register `init` command on root.
   - Flow:
     a. Load global config. If no instances, error with suggestion to run `auth add`.
     b. Get instance names from global config.
     c. Display instance picker TUI.
     d. Fetch application list from selected instance using HTTP GET to `{url}/api/v1/applications` with Bearer token (direct `net/http` call; full client is T11).
     e. Display application picker TUI.
     f. Display permission deny-list multi-select TUI.
     g. Validate deny list with `permissions.ValidateDenyList()`.
     h. Create `ProjectConfig` struct.
     i. Save to `.safe-ify.yaml` in current directory via `config.SaveProject()`.
     j. Print success message with summary.

3. Run `go build ./...` and verify success.
