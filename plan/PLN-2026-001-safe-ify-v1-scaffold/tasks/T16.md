## Instructions

1. Create `internal/audit/types.go`:
   - `type Entry struct` with fields: Timestamp (time.Time), Command (string), AppUUID (string), Instance (string), Result (string: "ok" or "error"), DurationMs (int64).
   - `(e Entry) String() string` -- formats as `YYYY-MM-DDTHH:MM:SSZ | command | app_uuid | instance | result | duration_ms`.

2. Create `internal/audit/logger.go`:
   - `type Logger struct` with `path string` field.
   - `NewLogger(path string) *Logger` -- if path empty, defaults to `~/.config/safe-ify/audit.log`.
   - `(l *Logger) Log(entry Entry) error`:
     a. Ensure directory exists (create with 0700 if needed).
     b. Open file with `os.OpenFile(path, os.O_APPEND|os.O_CREATE|os.O_WRONLY, 0600)`.
     c. Write `entry.String() + "\n"`.
     d. Close file.
   - `DefaultAuditLogPath() string` -- returns `~/.config/safe-ify/audit.log`.

3. Integrate audit logging into agent commands via a wrapper function:
   - Modify `internal/cli/agent.go`: create an audit middleware wrapper that all five agent commands (deploy, redeploy, logs, status, list) call.
   - Define the wrapper function:
     ```go
     func runAgentCommand(cmd *cobra.Command, commandName string, fn func(cfg *config.RuntimeConfig, client *coolify.Client) (interface{}, error)) error
     ```
   - The wrapper:
     a. Calls `resolveAgentConfig(cmd)` to get runtime config and client.
     b. Creates an `audit.Logger` using `audit.DefaultAuditLogPath()`.
     c. Records `start := time.Now()` before calling `fn(cfg, client)`.
     d. After `fn` returns, computes `duration := time.Since(start).Milliseconds()`.
     e. Builds `audit.Entry{Timestamp: start.UTC(), Command: commandName, AppUUID: cfg.AppUUID, Instance: cfg.InstanceName, Result: "ok" or "error", DurationMs: duration}`.
     f. Calls `logger.Log(entry)` (audit write errors are logged to stderr but do not fail the command).
     g. Returns the result or error from `fn`.
   - Modify each of the five agent command files (deploy.go, redeploy.go, logs.go, status.go, list.go) to call `runAgentCommand()` instead of directly calling the API. This ensures all five commands are audited uniformly.

4. Run `go build ./...` and verify success.
