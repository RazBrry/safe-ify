## Instructions

1. Initialize Go module: `go mod init github.com/erwinmaasbach/safe-ify` with Go 1.22+.
2. Create `Makefile` per 05-tech-spec-dev-ops.md section 1.1.
3. Create `cmd/safe-ify/main.go` as the entrypoint. It calls `cli.Execute()`.
4. Create `internal/cli/root.go`:
   - Define root Cobra command with `Use: "safe-ify"`.
   - Add global flags: `--json`, `--config`, `--project`, `--version`.
   - Wire `Execute()` function.
5. Create `internal/config/types.go` with all config struct types per 03-arch section 3 (`GlobalConfig`, `Instance`, `DefaultSettings`, `PermissionConfig`, `ProjectConfig`, `RuntimeConfig`).
6. Create `internal/config/global.go`:
   - `CheckPermissions(path)` -- validates file is 0600, directory is 0700.
   - `LoadGlobal(path)` -- checks permissions, reads YAML, unmarshals.
   - `SaveGlobal(path, cfg)` -- marshals YAML, creates dir with 0700, writes file with 0600.
   - `DefaultGlobalConfigPath()` -- returns `~/.config/safe-ify/config.yaml`.
7. Create `internal/config/project.go`:
   - `LoadProject(path)` -- reads and unmarshals project YAML.
   - `FindProjectConfig(startDir)` -- traverses parent directories looking for `.safe-ify.yaml` (per D10).
   - `SaveProject(path, cfg)` -- marshals and writes with 0644.
8. Create error types in `internal/config/errors.go` per 04-config-permissions section 5.
9. Run `go mod tidy` to fetch dependencies.
10. Verify `go build ./...` succeeds.
11. Verify `go vet ./...` passes.
