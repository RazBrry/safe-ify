## Instructions

1. Create `internal/permissions/types.go`:
   - Define `AllAgentCommands` slice: `["deploy", "redeploy", "logs", "status", "list"]`.
   - Define `PermissionDeniedError` with `Command` and `DeniedBy` fields.
   - `PermissionDeniedError` implements `error` with message: `"command '<cmd>' is not permitted for this project (denied by <source>)"`.

2. Create `internal/permissions/enforcer.go`:
   - `type Enforcer struct` with `allowed map[string]bool` field.
   - `NewEnforcer(global config.GlobalConfig, project config.ProjectConfig) *Enforcer`:
     a. Start with all `AllAgentCommands` allowed.
     b. Apply global `Defaults.Permissions.Deny` -- set matching commands to false.
     c. Apply project `Permissions.Deny` -- set matching commands to false.
     d. Return Enforcer with computed map.
   - `(e *Enforcer) Check(command string) error`:
     a. If `allowed[command]` is false, return `PermissionDeniedError`.
     b. Otherwise return nil.
   - `(e *Enforcer) AllowedCommands() []string` -- returns list of allowed command names.
   - `(e *Enforcer) DeniedCommands() []string` -- returns list of denied command names.
   - `ValidateDenyList(deny []string) error` -- rejects unknown command names.

3. Verify `go build ./...` succeeds.
