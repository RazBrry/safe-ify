## Instructions

1. Create `internal/permissions/enforcer_test.go`:
   - `TestResolvePermissions_NoRestrictions` -- no denials in global or project, all 5 commands allowed.
   - `TestResolvePermissions_GlobalDeny` -- global denies "deploy", verify deploy not allowed, others allowed.
   - `TestResolvePermissions_ProjectDeny` -- project denies "logs", verify logs not allowed, others allowed.
   - `TestResolvePermissions_ProjectCannotEscalate` -- global denies "deploy", project deny list is empty, verify deploy STILL denied. This is the critical test.
   - `TestResolvePermissions_CombinedDenials` -- global denies "deploy", project denies "logs", verify both denied, others allowed.
   - `TestEnforcer_AllowedCommand` -- Check("status") on unrestricted enforcer returns nil.
   - `TestEnforcer_DeniedCommand` -- Check("deploy") on enforcer with deploy denied returns PermissionDeniedError.
   - `TestEnforcer_DeniedBy_Global` -- verify DeniedBy field says "global" when globally denied.
   - `TestEnforcer_DeniedBy_Project` -- verify DeniedBy field says "project" when project-denied.
   - `TestValidateDenyList_ValidCommands` -- all 5 valid names accepted.
   - `TestValidateDenyList_InvalidCommand` -- "delete" rejected.
   - `TestValidateDenyList_EmptyList` -- empty list accepted.

2. Run tests: `go test ./internal/permissions/ -v -race`
3. Verify all tests pass.
