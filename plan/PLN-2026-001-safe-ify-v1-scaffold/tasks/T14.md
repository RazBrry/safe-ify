## Instructions

1. Create `internal/coolify/client_test.go`:
   - Setup helper: `newTestServer(handler)` creates `httptest.NewServer` and returns a `Client` pointing to it.
   - `TestClient_Healthcheck_Success` -- mock returns 200 "OK", verify no error.
   - `TestClient_Healthcheck_AuthFailure` -- mock returns 401, verify error.
   - `TestClient_ListApplications_Success` -- mock returns JSON array of applications, verify parsed correctly.
   - `TestClient_GetApplication_Success` -- mock returns single application JSON, verify fields.
   - `TestClient_GetApplication_NotFound` -- mock returns 404, verify error message mentions UUID.
   - `TestClient_Deploy_Success` -- mock returns deploy response JSON, verify deployment UUID extracted.
   - `TestClient_Deploy_RateLimit` -- mock returns 429 with Retry-After header, verify error includes retry info.
   - `TestClient_Restart_Success` -- mock returns 200, verify no error.
   - `TestClient_GetLogs_Success` -- mock returns log lines, verify count and content.
   - `TestClient_GetLogs_TailParameter` -- verify tail query parameter sent to server.
   - `TestClient_NetworkError` -- client points to closed server, verify network error.
   - `TestClient_BearerToken` -- verify Authorization header set on all requests.

2. Create `internal/cli/output_test.go`:
   - `TestOutputJSON_Success` -- verify `{ok: true, data: ..., error: null}` structure.
   - `TestOutputJSON_Error` -- verify `{ok: false, data: null, error: {code, message}}` structure.
   - `TestOutputJSON_ValidJSON` -- verify output is valid JSON (parseable).

3. Run all tests: `go test ./internal/coolify/ ./internal/cli/ -v -race`
4. Verify all tests pass.
