## Instructions

1. Create `internal/coolify/types.go`:
   - `Application` struct with fields: UUID, Name, Description, FQDN, Status, BuildPack, GitRepository, GitBranch, CreatedAt, UpdatedAt. Use `json:` struct tags matching Coolify API response (see 08-api-matrix section 6).
   - `DeployResponse` struct: Deployments array with Message, ResourceUUID, DeploymentUUID (see 08-api-matrix section 7).
   - `LogsResponse` struct: array of log line strings.
   - `CoolifyError` struct with StatusCode, Message, Raw fields. Implements `error`.

2. Create `internal/coolify/client.go`:
   - `type Client struct` with `baseURL string`, `token string`, `httpClient *http.Client`.
   - `NewClient(baseURL, token string) *Client` -- creates client with 30s timeout, sets User-Agent `safe-ify/1.0`.
   - Private helper `doRequest(ctx, method, path, query) (*http.Response, error)` -- builds request, sets Bearer token header, executes, handles non-2xx responses by returning `CoolifyError`.
   - Private helper `handleError(resp) error` -- maps HTTP status codes to user-friendly messages per 08-api-matrix section 5 (401 -> suggest auth add, 404 -> check UUID, 429 -> show retry-after).

3. Create `internal/coolify/applications.go`:
   - `(c *Client) Healthcheck(ctx context.Context) error` -- GET `/api/v1/healthcheck`, return nil on 200.
   - `(c *Client) Version(ctx context.Context) (string, error)` -- GET `/api/v1/version`.
   - `(c *Client) ListApplications(ctx context.Context) ([]Application, error)` -- GET `/api/v1/applications`, unmarshal.
   - `(c *Client) GetApplication(ctx context.Context, uuid string) (*Application, error)` -- GET `/api/v1/applications/{uuid}`.
   - `(c *Client) GetLogs(ctx context.Context, uuid string, tail int) ([]string, error)` -- GET `/api/v1/applications/{uuid}/logs?tail={tail}`.

4. Create `internal/coolify/deploy.go`:
   - `(c *Client) Deploy(ctx context.Context, uuid string, force bool) (*DeployResponse, error)` -- GET `/api/v1/deploy?uuid={uuid}&force={force}`.
   - `(c *Client) Restart(ctx context.Context, uuid string) error` -- GET `/api/v1/applications/{uuid}/restart`.

5. Run `go build ./...` and verify success.
