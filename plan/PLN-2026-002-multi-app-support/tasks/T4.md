## Instructions

1. **Create/update `internal/config/project_test.go`** with the following tests:
   - `TestLoadProject_MultiApp`: Write a multi-app YAML to a temp file, load it, verify Apps map has correct entries and AppUUID is empty.
   - `TestLoadProject_Legacy`: Write a legacy YAML (with `app_uuid`), load it, verify normalization to Apps map with key "default".
   - `TestLoadProject_BothFormats`: Write YAML with both `app_uuid` and `apps`, verify error returned.
   - `TestLoadProject_EmptyApps`: Write YAML with empty `apps:` map, verify error returned.
   - `TestLoadProject_InvalidAppName`: Write YAML with app name containing spaces/special chars, verify error returned.
   - `TestLoadProject_MissingAppUUID`: Write YAML with app entry having empty `uuid`, verify error returned.

2. **Create/update `internal/config/runtime_test.go`** with the following tests:
   - `TestResolveRuntime_SingleApp_NoFlag`: Single-app config, empty appName, verify resolves correctly.
   - `TestResolveRuntime_MultiApp_WithFlag`: Multi-app config, appName="api", verify correct UUID returned.
   - `TestResolveRuntime_MultiApp_NoFlag`: Multi-app config, empty appName, verify AppAmbiguousError.
   - `TestResolveRuntime_AppNotFound`: appName="unknown", verify AppNotFoundError.
   - `TestResolveRuntime_ThreeLayerDeny`: Set global deny=["deploy"], project deny=["redeploy"], app deny=["logs"]. Verify all three are denied and the rest are allowed.

3. **Update `internal/permissions/enforcer_test.go`** with:
   - `TestEnforcer_WithAppDeny`: Create enforcer with app deny list, verify additional commands denied, deniedBy="app".
   - `TestEnforcer_AppDenyDoesNotEscalate`: Deny "deploy" globally, pass empty app deny. Verify still denied.

4. **Run all tests:** `go test ./internal/config/ ./internal/permissions/`

5. **Run full suite:** `go test ./...` to verify no regressions.
